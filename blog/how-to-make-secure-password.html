How to Make a Secure Password

Having a secure password is vital to keeping your account safe. The question is, how do you make a secure password? Well, we know that we need:
  • 10+ characters (15+ best)
  • Letters (Uppercase and lowercase)
  • Numbers
  • Special characters (!?/@#$%^&, etc)
  • A variety of characters

However, keep in mind that while a long or random password would be secure, like “dh!$Zt6kl%Vw@cxgvt1lz8s,7”, which would be very secure, chances are, you probably won’t be able to remember it.
Another important thing to know is to avoid using the same password on multiple accounts. If a hacker has gained access to a password for one account, they would likely try the same password (or slight variations) on other accounts. This could lead to multiple accounts being hacked, which would be catastrophic. Instead, have different passwords to make your accounts more secure.
