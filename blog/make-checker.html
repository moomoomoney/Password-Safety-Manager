How Did You Make the Password Security Checker?



How it Works

This password checker uses an 8-category grading system to rate the secureness of your password.



If your password is "insecure" or "very insecure", then you should try to make it more secure by following the tips in our blog. However, don't make it too complicated or you will struggle to remember it. If your password is "secure" or "very secure", you have a good password.

How We Made It

First, we take the input (your password) from the input box. After that, the input box value is passed through a variety of functions. These functions take in the input of the password, and they output the secureness score in that category.
